</br>logmrgr
</br>Приложение для объединения лог-файлов по времени событий в них.
</br>Суперлог.

`logmrgr 021` — обрабатывает логи по частям, не загружая всё в память, что позволяет работать с файлами любого размера без зависаний. Используется многопроцессорная обработка и сортировка через `heapq`, обеспечивая точный хронологический порядок. Добавлены корректный прогресс, защита от ошибок в GUI, автоматическая очистка временных файлов и стабильное сохранение.

## Использование
1. Загрузите `.log` или `.json` файлы.
2. Или загрузите сразу папку с такими файлами.
3. Соедините логи.
4. Сохраните результат.
